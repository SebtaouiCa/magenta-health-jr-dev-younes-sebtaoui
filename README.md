
1. What is one developer tool or website you use regularly? Why?

I regularly use Chrome DevTools because it helps me quickly debug JavaScript, check console errors, and inspect network requests. It’s very useful to understand what’s happening in the browser and fix issues faster.


2. If your code isn’t working, what’s your first step to debug it?
 
My first step is to check the console for errors. Then I use console.log to see what values I’m getting and where the problem starts. If needed, I simplify the code to isolate the issue and fix it step by step.
